⌨ 입력장치

입력장치는 사용자로부터 데이터를 입력받는 장치이다.
  • 키보드
  • 마우스
  • 스캐너
  • 마이크
  • 웹캠

🖨 출력장치

출력장치는 처리 결과를 사용자에게 제공한다.
  • 모니터
  • 프린터
  • 스피커
  • 3D프린터

🛠 시스템소프트웨어

컴퓨팅 시스템 자체의 동작을 관리하고 제어하는 소프트웨어로, 운영체제와 유틸리티 프로그램으로 구분된다.

📱 응용소프트웨어

게임, 웹브라우저, 문서작성 프로그램 등이 있다.

⚙ 컴퓨팅 시스템 동작 원리

입력 → 처리 → 기억 → 출력 → 제어 과정으로 동작한다.

🔄 컴퓨팅 시스템 동작 과정

전원공급 → 기본 입출력 검사 → 부트 로더 실행 → 운영 체제 로드 → 초기화 및 시스템 설정 → 사용자 인터페이스 제공
